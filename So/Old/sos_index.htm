Socialization Scale (SOS) Resources

The SOS scale includes a total score.

Data Entry Instructions

For each participant, enter the following information:

Column 1 = Participant ID#

Columns 2-55 = Item scores for items 1-54; enter "T" for each item answered "true" and "F" for each item answered "false".

For missing values, enter "MISS".

Scoring Output Abbreviations

SOS_TOT Total Score
